CSS position 속성 시각 예제

static : 기본값. 문서 흐름에 따라 배치됨

relative : 원래 위치를 기준으로 top, left 등으로 이동 (공간은 유지됨)

absolute : 가장 가까운 position이 지정된 조상을 기준으로 위치

fixed : 브라우저 화면(Viewport)을 기준으로 고정됨

sticky : 스크롤 시 지정된 위치에 도달하면 고정됨 (relative + fixed의 성질)
